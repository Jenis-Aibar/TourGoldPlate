TourGoldPlate — плагин для Paper, добавляющий  золотую плиту, которая даёт игроку деньги каждый тик, пока он стоит на нем.

Возможности
> Настройка одной золотой плиты через команду.
> Автоматическая выдача денежной награды каждые N тиков.
> Полная поддержка Vault и совместимость с большинством плагинов экономики.
> Только один игрок может владеть плитой одновременно.
> Настраиваемый размер награды и интервал выдачи через config.yml.

Требования:
  Paper 1.21+
  Vault 1.7.3
